About KryptoSide

Welcome to KryptoSide, your one-stop destination for comprehensive cryptocurrency data and statistics. In an ever-evolving digital economy, staying updated with the latest trends, prices, and insights into the cryptocurrency market is essential. KryptoSide provides a user-friendly platform where you can access real-time data, detailed analytics, and historical trends for a wide range of cryptocurrencies.

KryptoSide is my final foundation-level portfolio project at the ALX Africa Software Engineering Program. It was inspired by my desire to try a few of what I have learnt during the program. I have always wanted to know how to use API for fetching and plotting data. I also wanted to be able to fetch data from a database and display on a web template. These I have achieved using KryptoSide. Althought the project is still in its infant stages. I hope to upgrade it further with features and make it a web service that people can truly use.
